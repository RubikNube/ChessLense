# ChessLense

ChessLense is a local chess analysis tool that uses Stockfish to evaluate positions. It consists of a Node.js backend that interfaces with Stockfish and a React frontend that provides an interactive chessboard for users to input positions and view analysis results.

## Development

Use `./dev.sh` from the project root to start the server and frontend
concurrently for local development.
